# Focus Pomodoro 

Focus Pomodoro Study Timer is a Chrome/Browser extension designed to help you stay focused and productive. It features a customizable Pomodoro timer and the ability to block distracting websites while the timer is active.

## Features
- Customizable Pomodoro timer to manage your work and break intervals.
- Website blocking during active focus sessions to eliminate distractions.
- Simple and intuitive interface.

## Installation
1. Download or clone this repository.
2. Open your browser and navigate to `chrome://extensions/`.
3. Enable "Developer mode" in the top-right corner.
4. Click "Load unpacked" and select the extension folder.

## Usage
1. Set your desired focus and break intervals in the timer settings. (delayInMinutes variable in popup.js).
2. Add websites to the blocklist that you want to restrict during focus sessions. (sites variable in popup.js)
3. Start the timer and stay focused!
